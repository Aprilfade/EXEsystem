## 使用方法

### 重新初始化数据
如果需要重新初始化数据，可以执行以下命令：

```bash
# 1. 初始化科目数据
mysql -uroot -p密码 -h localhost --default-character-set=utf8mb4 < 初始化科目数据.sql

# 2. 初始化题库数据
mysql -uroot -p密码 -h localhost --default-character-set=utf8mb4 < 初始化题库数据.sql
```

### 生成新的题库数据
如果需要生成不同的题目集合，可以运行：

```bash
python generate_questions.py
```

这将重新生成 `初始化题库数据.sql` 文件，题目内容会有所不同（因为使用了随机生成）。

## 注意事项
1. 数据库名称为 `exam_system`
2. 执行SQL时必须使用 `--default-character-set=utf8mb4` 参数以支持中文
3. 体育、音乐、美术科目暂未生成题目（可以后续扩展题目模板）
4. 所有题目的难度系数、选项、答案都是随机生成的，实际使用时需要根据具体需求进行调整
