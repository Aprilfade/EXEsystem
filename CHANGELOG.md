## [3.06] - 2026-03-06

### ✨ 新增功能

#### 🤖 AI监控增强
- **CSV导出功能**
  - AI调用日志CSV格式导出
  - 支持时间范围筛选
  - 自动处理CSV特殊字符转义
  - UTF-8 BOM支持，Excel直接打开无乱码
  - 包含完整调用信息：用户ID、功能类型、AI提供商、响应时间、Token使用量、成本等

#### 🎓 智能推荐系统
- **基于内容的课程推荐**
  - 分析用户最近3个月考试成绩
  - 按科目统计平均得分率
  - 智能推荐薄弱科目的课程
  - 推荐分数算法：考虑薄弱程度和考试次数置信度
  - 自动生成个性化推荐理由
  - 当无考试数据时，返回热门课程

#### 🔍 搜索功能优化
- **课程搜索实现**
  - 支持课程名称和描述搜索
  - 相关度计算算法（名称权重80%，描述权重20%）
  - 按相关度自动排序
  - 限制返回Top 10结果

### 🐛 修复
- **用户认证优化**
  - AI试卷生成获取真实用户ID（不再使用临时ID）
  - 从SecurityContext正确获取当前登录用户
  - 完善用户不存在的错误处理

- **登出流程优化**
  - 教师端退出登录后跳转到Portal页面
  - 与学生端保持一致的退出体验
  - 用户可重新选择登录角色

### 🔧 技术改进
- **代码质量提升**
  - 完成所有TODO项的实现
  - 优化CSV字段转义逻辑
  - 改进课程推荐算法
  - 增强错误处理和日志记录

### 📚 文档
- 完整文档体系整理
- API接口文档更新
- 功能实现文档完善

## 版本说明

### 版本号规则

采用 `主版本号.次版本号.修订号` 格式：

- **主版本号**：重大架构变更或不兼容的 API 修改
- **次版本号**：新功能添加（向后兼容）
- **修订号**：Bug 修复和小优化

### 变更类型

- `✨ 新增` (Added) - 新功能
- `🐛 修复` (Fixed) - Bug 修复
- `🔧 重构` (Changed) - 功能变更
- `🗑️ 删除` (Removed) - 移除功能
- `⚡ 性能优化` (Performance) - 性能改进
- `📚 文档` (Documentation) - 文档更新
- `🔐 安全` (Security) - 安全相关
- `🎨 改进` (Improved) - 用户体验改进
